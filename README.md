# ✈️ Saudia Airlines — Booking Analytics Dashboard

An interactive analytics dashboard built entirely in Microsoft Excel
to visualize Saudia Airlines booking data.

## 📊 Pages
- **Overview** — KPIs, channel performance, trip types,
  daily volume & peak hours
- **Services** — Baggage usage, extra seats & meal demand
- **Heat Map** — Global booking insights by country

## 🔍 Key Metrics
- Total Bookings: 49,281
- Completed Bookings: 7,391
- Avg Purchase-Led: 2
- Baggage Usage: 67% | Extra Seats: 30% | Meal Demand: 43%

## 🛠️ Tools Used
- Microsoft Excel (Charts, Slicers, Pivot Tables)

## 📁 Dataset
Booking records including sales channel, trip type,
flight duration, booking origin & timestamps.
